# Tic Tac Toe Game

This is a simple Tic Tac Toe game implemented in JavaScript.

## Features

- Play against a friend or against the computer.
- Track game statistics such as wins, losses, and winning streaks.
- Responsive design for a seamless experience on different devices.

## How to Play

1. Clone this repository to your local machine.
2. Open `index.html` in your web browser.
3. Choose whether to play against a friend or against the computer.
4. Enjoy the game!
